Detailed Description

The main window of app. Holds the three big parts.

1) Left menu widget - parts of book picker. Graphs algorithms, sorting algorithms and their settings.

2) Middle viewport widget - show Qt3D objects.

3) Right menu widget - show current properties of scene presenting in viewport.
